# Code of Conduct

This project adopts the **Contributor Covenant, version 2.1**.

The full text — including standards of behavior, enforcement
responsibilities, scope, and enforcement guidelines — is published at
https://www.contributor-covenant.org/version/2/1/code_of_conduct/ and
applies to this project in full.

## Reporting

Report violations to **veli-matti@vanamo.com** with subject
`[conduct] <summary>`. Reports are confidential; we respond within 72
hours.

For security vulnerabilities in the skill itself (not community
conduct), see [SECURITY.md](SECURITY.md) — different process.

## Scope

Applies to every interaction related to this project: GitHub issues,
pull requests, discussions, commit messages, release notes, and any
communication channel where contributors represent the project.
